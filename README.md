# 符号说明
各类注释符号为VSCode的Colorful Comments插件功能获取，相关颜色定义如下：
* #* : 特别个人注释说明
* #^ : 特别值得注意或学习的说明
* #! : 特别警告说明
* #~ : 相关可参考的连接

其他相关定义当前还未使用。
